# 1. Installazione degli Strumenti Necessari

Per utilizzare Git-crypt, è necessario avere installato Git, GPG e Git-crypt stesso.

## 1.1 Git
Assicurati che Git sia installato e configurato.

## 1.2 GPG (GNU Privacy Guard)
GPG è il gestore delle chiavi crittografiche.

* **macOS (Homebrew):** `brew install gpg`
* **Linux (Debian/Ubuntu):** `sudo apt-get install gpg`
* **Windows:** Usa [Gpg4win](https://www.gpg4win.org/) o installa GPG tramite [WSL (Windows Subsystem for Linux)](https://docs.microsoft.com/it-it/windows/wsl/install).

## 1.3 Git-crypt
Installazione dell'estensione Git.

* **macOS (Homebrew):** `brew install git-crypt`
* **Linux (Debian/Ubuntu):** `sudo apt-get install git-crypt`
* **Altro:** Consulta il [repository ufficiale Git-crypt](https://github.com/AGWA/git-crypt).
